# 🛒 ShopKart E-Commerce Analytics Dashboard

## About the Project

This is an E-Commerce Analytics project that I built using SQL and Power BI. The aim of this project was to understand how sales data can be analyzed and converted into meaningful business insights using interactive dashboards.

I first worked with the dataset in SQL to create the database and write queries for analysis. After that, I imported the data into Power BI, built relationships between tables, created DAX measures, and designed interactive dashboards.

## Dashboard Overview

The dashboard provides insights into:

- Total Sales
- Total Orders
- Total Quantity Sold
- Average Order Value
- Sales by Category
- Sales by City
- Top Products
- Top Customers
- Payment Method Analysis
- Monthly and Daily Sales Trends

The report is divided into three pages:

- Executive Dashboard
- Sales Dashboard
- Customer Dashboard

## How to Use

1. Open the SQL file and create the database.
2. Import the provided datasets.
3. Run the SQL queries if you want to practice.
4. Open the `.pbix` file in Power BI Desktop.
5. Explore the interactive dashboards.
